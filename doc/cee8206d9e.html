Methods

process

Process the the token with the given processer object. Runs a search on the token's content based on the given processer's regExp and if a match is found, calls the module:model/texttools/processabletools.Processing.Processer#onMatch method back with the results.

It could find matches only inside of the leaf tokens interval, cross-border matches remains unnoticed!

replace

Replace substings in the token's content with the return value of the given replacer object's replace method. Runs a search on the token's content based on the given replacer's regExp and if a match is found, calls the module:model/texttools/processabletools.Processing.Replacer#replace method back with the results. It uses the string.replace method

It could find matches only inside of the leaf tokens interval, cross-border matches remains unnoticed!
