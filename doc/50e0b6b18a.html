Class: Interval

model/texttools/processabletools.Tokens. Interval

Represents an open interval.
Since the interval's open, it's endpoints are excluded from it! Therefore interval ( 0, 1 ) and ( 1, 2 ) doesn't intersect!

A numerical representation of the relation.

  • -1: the given interval intersects the starting endpoint of the owner
  • 0: the given interval is inside the owner
